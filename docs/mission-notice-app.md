# 미션 알림장 실행 안내

아이들이 매일 알림장 미션을 완료하고 보상 코인으로 캐릭터를 성장시키거나 꾸밀 수 있는 게임형 알림장 앱 프로토타입입니다.

## 주요 기능

- 과목, 숙제/공지, 준비물을 직접 입력하는 알림장 작성
- 오늘의 알림장 미션 완료/취소
- 미션 보상 코인 지급
- 하루 미션 완료율과 캐릭터 레벨 표시
- 코인으로 꾸미기 아이템 구매 및 캐릭터 즉시 반영

## 실행 방법

아래 명령으로 로컬 서버를 실행한 뒤 브라우저에서 <http://127.0.0.1:5173>을 열어주세요.

```bash
npm install
npm run start
```

`index.html` 파일을 직접 열어도 동작하도록 CSS/JS 경로는 상대 경로로 연결되어 있습니다.

## 빌드/검증

```bash
npm run build
```
